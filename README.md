# CHIP 8 Emulator

This is an implementation of https://en.wikipedia.org/wiki/CHIP-8 

the ideia is run games for CHIP8. 


## Requirements

* [dot net core](https://dotnet.microsoft.com/download)

## Usage

1. Navegate to the folder 
2. Run `dotnet run`
